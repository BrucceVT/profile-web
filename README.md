## About The Project

![Product Screenshot](https://placehold.co/600x400?text=Portfolio+Screenshot)

This project is a high-performance, interactive portfolio website designed to demonstrate modern web development capabilities. It leverages the power of **React 19**, **Three.js** for 3D visualizations, and **Framer Motion** for seamless animations, all powered by the ultra-fast **Bun** runtime.

Key features include:
*   **Immersive 3D Experience**: Integrated 3D models and scenes using React Three Fiber.
*   **Responsive & Adaptive**: Fully responsive design optimized for all device sizes via Tailwind CSS 4.
*   **Global State Management**: robust context-based state management for a fluid user experience.
*   **Internationalization**: Built-in support for multiple languages (i18n).
*   **Modern Architecture**: Feature-based folder structure for scalability and maintainability.

## Getting Started

To get a local copy up and running follow these simple steps. This project uses **Bun** as the package manager and runtime.

### Prerequisites

*   **Bun** (v1.0 or later)
    ```sh
    # Windows (PowerShell)
    powershell -c "irm bun.sh/install.ps1 | iex"
    
    # Linux / macOS
    curl -fsSL https://bun.sh/install | bash
    ```

### Installation

1.  Clone the repository
    ```sh
    git clone https://github.com/BrucceVT/profile-web.git
    ```
2.  Navigate to the project directory
    ```sh
    cd profile-web
    ```
3.  Install dependencies
    ```sh
    bun install
    ```
4.  Start the development server
    ```sh
    bun run dev
    ```
